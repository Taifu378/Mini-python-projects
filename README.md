# 🐍 Python Mini Projects Collection

A collection of functional Python scripts exploring basic logic, user input handling, and the `random` library. This repository serves as a portfolio of beginner-friendly utilities and terminal-based games.

## 🚀 Projects Overview

| Project Name | Description | Key Features |
| :--- | :--- | :--- |
| **Password Generator** | Creates secure, randomized passwords based on user-defined criteria. | `random.shuffle`, character selection |
| **Rock Paper Scissors** | A classic terminal game where you compete against a randomized computer opponent. | ASCII art, conditional logic |
| **Treasure Island** | A choice-based text adventure game with multiple paths and outcomes. | Nested logic, ASCII graphics |
| **Tip Calculator** | A practical tool to calculate bill splits and tip percentages. | Data type conversion, rounding logic |

## 🛠️ Getting Started

### Prerequisites
Ensure you have **Python 3.x** installed. You can check your version by running:
```bash
python --version
```

### Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/mini-python-projects.git
   cd mini-python-projects
   ```

2. **Run a script:**
   Navigate to the repository and run any file directly using Python:
   ```bash
   python Passwordgenerator.py
   ```

📂 Detailed Script Descriptions

🔑 Password Generator
Generates a strong password by asking the user for a specific number of letters, symbols, and numbers. It uses the `random.choice` method to pick characters and `random.shuffle` to ensure the final string is not in a predictable order.

✊ Rock Paper Scissors
An interactive game where the player inputs `0` for Rock, `1` for Paper, or `2` for Scissors. The script compares the player's choice against a `random.randint(0, 2)` generated by the computer and displays the result using ASCII art.

🏝️ Treasure Island Game
A text-based quest where your goal is to find the treasure. Players must navigate through choices—choosing to turn left or right, swim or wait, and picking between colored doors—to avoid "Game Over" scenarios.

#💸 Tip Calculator
Takes a total bill amount, a desired tip percentage (10, 12, or 15), and the number of people splitting the bill. It calculates and rounds the final amount each person needs to pay to two decimal places.
